# Classic Tic Tac Toe

A premium, elegantly designed classic Tic Tac Toe game built using vanilla web technologies (HTML, CSS, JavaScript).

## 🌟 Features

*   **Classic Aesthetic:** A clean, minimalist design inspired by traditional printed games, featuring soft off-white backgrounds and crisp typography.
*   **Premium Typography:** Utilizes Google Fonts (`Playfair Display` for headings and scores; `Inter` for general text) for a polished, highly readable interface.
*   **Interactive Gameplay:** Smooth, click-to-place mechanics with elegant hover effects and subtle fade-in animations for game pieces.
*   **Live Scoreboard:** Automatically tracks wins for "Player X", "Player O", and "Ties" across multiple rounds.
*   **Responsive Layout:** Adapts seamlessly to different screen sizes, ensuring a great experience on both desktop and mobile devices.
*   **No Dependencies:** Built entirely with vanilla HTML, CSS, and JavaScript. No frameworks or external libraries required.

## 🛠️ Technologies Used

*   **HTML5:** Semantic structuring of the game board and UI.
*   **CSS3:** Styling, grid layouts, custom variables, hover effects, and keyframe animations.
*   **JavaScript (ES6):** Game logic, state management, event handling, and DOM manipulation.

## 🚀 Getting Started

It's incredibly easy to run this project! Since it consists of static files, you don't need a build process or a server.

1.  **Download or Clone** the repository/files to your local machine.
2.  **Open `index.html`** in your preferred web browser (Chrome, Firefox, Safari, Edge, etc.). You can do this by double-clicking the file or dragging it directly into an open browser window.
3.  **Play!** Player X always goes first.

## 🎮 How to Play

1.  The game starts with an empty 3x3 grid.
2.  **Player X** clicks any empty cell to place their mark.
3.  **Player O** then takes their turn by clicking an empty cell.
4.  The goal is to get 3 of your marks in a row (horizontally, vertically, or diagonally).
5.  If all 9 cells are filled and no player has 3 in a row, the game ends in a tie.
6.  The scoreboard updates automatically at the end of each round.
7.  Click the **"Restart Game"** button at any time to clear the board and start a new round (scores are kept intact).
